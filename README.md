# Libraries Required
You will need:
- glfw
  - Run these commands below: 

```
sudo apt-get install libglfw3
sudo apt-get install libglfw3-dev
```

- libjsoncpp-dev
  - Run these commands below:

```
sudo apt-get install libjsoncpp-dev
sudo ln -s /usr/include/jsoncpp/json/ /usr/include/json
```
